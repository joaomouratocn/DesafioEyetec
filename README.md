# Desafio-Eyetec

Projeto proposto para prova de conhecimento técnico da empresa Eyetec.

## Tecnologias
- **Kotlin**: Linguagem recomendada pela Google para apps nativos.
- **ViewBinding**: Utilizado para acessar os componentes do layout de maneira simples, deixando o SDK do Android cuidar destas ligações.
- **Navigation Component**: Para simplificar a navegação entre as telas do app, ajudando na passagem de parâmetros e na pilha de navegação.
- **MVVM**: Arquitetura que separa a lógica (ViewModel) da interface de usuário (Fragment), separando as responsabilidades de camada.
- **Koin**: Framework de Injeção de Dependência escolhido por usar o Kotlin como linguagem e pela simplicidade na configuração.
- **Room Database**: Utilizado para persistência local de dados. O mais adequado e recomendado em ambientes Android nativo.
- **CameraX**: API do Jetpack que simplifica o uso da camera do dispositivo.
- **ML Kit (Face Detection)**: Utilizado para processamento de imagem em tempo real, realizando a validação do rosto do usuário.

## Como rodar o projeto

Desenvolvi este projeto utilizando o Android Studio Ladybug, recomendo o mesmo para evitar incompatibilidade de versão, 
mas o APK do app esta na pasta apk, na raiz projeto.
